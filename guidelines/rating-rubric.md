# Rating Task Rubric

## Overview

In **rating tasks**, you evaluate a single AI response against defined quality dimensions. There is no comparison — you assess the response on its own merits.

---

## Step-by-Step Process

1. **Read the prompt** — Understand the user's intent and what a good response should include.
2. **Read the response carefully** — Look for both strengths and weaknesses.
3. **Rate each dimension independently** — Don't let one strong/weak area influence your other ratings.
4. **Calibrate your scores** — Use the full scale. Not everything is a 3 or 4.
5. **Write your justification** — Support your ratings with evidence from the response.

---

## Rating Scale (1–5)

| Score | Meaning |
|-------|---------|
| 1     | **Unacceptable** — Response is wrong, harmful, or completely unhelpful |
| 2     | **Below average** — Significant problems that undermine usefulness |
| 3     | **Average** — Meets basic expectations with some gaps |
| 4     | **Above average** — Good quality with only minor issues |
| 5     | **Excellent** — Would be hard to meaningfully improve |

---

## Calibration Guidelines

Ask yourself these questions for each score:

- **Would I send this to a colleague?** (if yes, likely 4+)
- **Does it contain any factual errors?** (if yes, cap accuracy at 3)
- **Could following this advice cause problems?** (if yes, cap safety at 2)
- **Would a beginner understand this?** (if relevant, factor into clarity)

---

## Special Considerations for Code Responses

- Does the code compile/run?
- Are there security vulnerabilities?
- Is error handling present?
- Are variable names meaningful?
- Is the code idiomatic for its language?

## Special Considerations for Creative Responses

- Is the writing engaging?
- Does it match the requested tone/style?
- Is it original or formulaic?
- Does it show understanding of the genre?

---

## Justification Requirements

Your justification should:
1. Mention the **strongest** aspect of the response
2. Mention the **weakest** aspect
3. Reference **specific content** (quote or paraphrase)
4. Be **at least 2 sentences**
